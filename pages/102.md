### The Janus Queue
 The Janus queue (installed with  `pip install janus` ) provides a solution for communication between threads and coroutines. In the Python standard library, there are two kinds of queues: `queue.Queue` A  *blocking*  queue, commonly used for communication and buffering between threads `asyncio.Queue` An  `async` -compatible queue, commonly used for communication and buffering between coroutines Unfortunately, neither is useful for communication between threads and coroutines! This is where Janus comes in: it is a single queue that exposes both APIs, a blocking one  *and*  an async one.
